Stack Docker activo

Este contenedor es un marcador de posición. Para ver tu app Vite/React, configurá en la raíz del API un archivo .env (o variables de entorno) con la ruta a tu front:

  • FRONTEND_BUILD_CONTEXT — carpeta del proyecto front (ej. ../arandano-web)
  • FRONTEND_DOCKERFILE — por defecto docker/front-build/Dockerfile
  • VITE_API_URL — URL del API desde el navegador (ej. http://localhost:3000)

Luego: docker compose --profile full up -d --build

API health: /health (puerto por defecto del stack).
